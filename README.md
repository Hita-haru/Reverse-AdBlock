# Reverse-AdBlock

## 説明
逆広告ブロッカーです。広告をブロックするのではなく、逆に表示します。（表示される広告は架空の広告です。）  
これはChrome拡張機能です。Google ChromeやMicrosoft Edge、Chromiumなどにインストールしてご利用ください。  
この拡張機能をインストールするとあらゆるサイトで広告が表示されます。どの広告がどの程度表示されるかはランダムです。  
広告によってwebページの一部が隠れ、その部分を視認、操作できなくなる可能性があります。十分ご注意ください。  

## 使い方
releasesからzipファイルを落としてchrome拡張機能をdeveloper modにしたあとそこにzipファイルをドラッグアンドドロップすれば入ります  
入ったらwebページを表示してみるともう広告が出てくるはずです  

## 免責事項
この拡張機能により表示される広告は架空のものです。実際の企業、団体、個人、物などとは一切関係ありません。
